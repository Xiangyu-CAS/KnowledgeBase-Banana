# AI Comic
我希望通过AI生成漫画内容，以下是一些相关的开发和使用说明。

- 人物知识库（已实现）
- 分镜脚本生成（AI 漫画页签中点击“生成分镜脚本”）
- 漫画图像生成（AI 漫画页签中点击“生成漫画页面”）

## 使用流程
1. 在“故事设定”中填写剧情概述，可使用 `@人物名` 引用知识库角色。
2. 选择分镜格数、画面风格，并填写页眉标题/页码（可选）。
3. 先生成分镜脚本，确认对白与镜头后再生成漫画页面。

## 输出示例目标
- 单张纵向长图
- 4~6 格条漫分镜
- 中文对白气泡
- 页眉标题 + PAGE 页码（可选）




- 流程1：上传小说，默认文本 `assets/凡人修仙传 第五卷 名震一方 第七百三十六章 破阵大战（一）.txt`
- 流程2：根据文本内容自动提取分析人物、物品，构建知识库
- 流程3：生成分镜脚本，生成漫画图像，注意结合知识库人物形象一致性
